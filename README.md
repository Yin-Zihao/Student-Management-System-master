学管系统/王昶潭方向

项目简介
学管系统是面向高校的综合性数据管理平台，结合 教研、科研、人事 等核心业务需求，旨在解决学院数据分散、统计困难、信息孤立的问题。系统通过统一的数据汇聚与标准化处理，实现竞赛获奖、科研成果、项目经费、人事档案等信息的集中管理，并提供实时可视化展示与智能分析，支持学院在教学管理、科研发展和人事决策上的数字化转型。

功能特性
教研管理

学生竞赛获奖信息录入与统计（校级、省级、国家级）

蓝桥杯国赛、省赛数据管理

大创项目与教研奖励成果归档

科研管理

论文成果管理（SCI、CCF会议论文等）

科研奖励与科研项目经费管理

经费到账与使用情况统计，科研成果趋势分析

人事管理

教职工基本信息与档案管理（学历、职称、党派、岗位）

学院与学校人事记录统一管理

教师荣誉称号、兼职、创业情况记录

新入职与特聘教师信息管理

数据展示与分析

教研、科研、人事数据大屏可视化

竞赛获奖趋势、科研成果分布、教师课时与评教结果统计

学生成绩、学分、排名与毕业预警分析

技术栈
前端：Vue.js / React / ECharts / DataV

后端：Spring Boot / Python

数据库：MySQL / PostgreSQL + Redis

部署环境：Windows/Linux + Nginx
